# プロダクトのタイトル

## プロダクトの紹介
- 登山の記録を残すプロダクトを作りたかった。

- 機能について
- 右側の必要項目を入力写真を添付し保存をするとデータがサーバーに保存され、右側の項目に表示されます。
- ユーザーログイン機能を追加しました
- 写真はテーブルではなくプロダクトのimagesフォルダーに保存できるようにしました。
